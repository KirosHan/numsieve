# numsieve
## 中国联通扫号器
###### C#写的扫号器，因为需求很简单所以直接用之前写的shodan-API工具改了改

###### url为拉取json的链接，可以直接用Fiddler抓取

###### 为避免封IP  采用单线程

### 更新日志：
